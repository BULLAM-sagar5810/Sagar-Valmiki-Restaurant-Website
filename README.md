# 🍽️ Sagar Valmiki Restaurant Website

A modern and visually appealing **restaurant website** developed using **HTML & CSS**. This project showcases a complete front-end design for a restaurant, including menu display, gallery, chef profiles, and an order interface.

---

## 🚀 Features

* 🧭 Navigation Bar with multiple sections
* 📂 Side Menu for food categories
* 🍛 Food Items Display (Idly, Dosa, Puri, etc.)
* 👨‍🍳 Chef Profiles Section
* 🖼️ Gallery with food images and pricing
* 📝 Order Form UI
* 📍 About Us section with contact details
* ✨ Hover effects and modern UI design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts
* Material Symbols Icons
